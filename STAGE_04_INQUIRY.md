
# STAGE 04 - 문의 저장 및 문의관리

## 목표

랜딩페이지 문의를 DB에 저장하고 관리자에서 확인한다.

## 파일

/page/landing_inquiry_update.php

/adm/landing/inquiry_list.php

## 입력 항목

* 이름
* 연락처
* 문의내용
* 개인정보 동의

## 저장 테이블

landing_inquiries

## 관리자 기능

* 문의 목록
* 랜딩별 문의 구분
* 상담상태 변경
* 삭제
* 검색

## 상태값

new

contacted

completed

## 완료 기준

* 문의 저장 가능
* 관리자 확인 가능
* 랜딩별 구분 가능
* 모바일 정상
